# basis_universal

## Installation

```shell
npm install --save basis_universal_wasm
```

## API

```javascript
import {importBasisEncoder} from "basis_universal_wasm"

importBasisEncoder().then((module)=>{
    // do something
})
```

## Contributing

To build the project locally, run:

```
npm install
npm run build
```
